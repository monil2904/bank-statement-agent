## What you are building
A Bank Statement Auto-Tag & Metrics Agent for an Indian MSME lending hackathon (FlexiLoans Agentic Hackathon, Topic 3).

## What it does
1. Accepts a bank statement CSV upload
2. Classifies every transaction into 8 categories using Claude claude-sonnet-4-20250514 (batched, 20 rows per call)
3. Computes 4 underwriting metrics deterministically (ABB, BTO, bounce ratio, FOIR)
4. Detects anomalies using rule-based logic + LLM explanation
5. Returns a structured JSON result rendered in a React dashboard

## Stack — do not deviate
- Backend  : Python 3.11, FastAPI, pandas, anthropic SDK
- LLM      : claude-sonnet-4-20250514 ONLY — no other model
- Frontend : React 18 + Vite, plain fetch, no UI library
- Data     : Synthetic CSV generated by generator.py
- Deploy   : Local only — uvicorn port 8000, vite port 5173

## Rules for the coding agent
1. Follow BUILD_SEQUENCE.md in order. Do not jump ahead.
2. After each phase checkpoint, stop and confirm output before proceeding.
3. Never hardcode ANTHROPIC_API_KEY — always read from environment variable.
4. Never call the LLM inside metrics.py — that module is pure math only.
5. The master prompt in MASTER_PROMPT.md is the source of truth — copy it exactly into classifier.py as SYSTEM_PROMPT.
6. If a checkpoint fails, fix that phase before moving to the next one.
7. All Pydantic models live in models.py — do not define inline schemas elsewhere.
8. CORS is enabled for localhost:5173 only.

## Environment setup (run once)
```bash

```
